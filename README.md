# Hey_Electra
ESP8266 ESP8285 ESP32 mDNS Wifi Relais Switch with switched time settings

Easy Home Automation

Tzapu wifimanager for wifi credentials dot.local mdns adres and i/o config

No settings hardcoded on first startup connect wifi direct to esp8266 AP=AccessPoint

Browse to 192.168.4.1

Configure and browse to http://?????.local adres wich you have just set

Android Chrome does not support mdns use bonjourbrowser app

No external apps or services needed (4 android bonjour app & NTP time)

Android BonjourBrowser http://wellenvogel.de/software/index.html

each device automaticly scanned linked URL list of all other devices in local network

no IP numbers but human friendly logical http://*.local addresses like http://living.local ,http://kitchen.local etcetera
